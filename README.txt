
Swastik Construction - Material Order Prototype (Offline web app)
Files:
- index.html : Single-file prototype. Open this file in your mobile or PC browser.
  - Supervisor tab: Fill site, material, qty, supplier and submit.
  - Admin tab: View orders, approve, mark delivered, export CSV.
How it works:
- Orders are saved in browser localStorage under key 'swastik_orders_v1'.
- To persist across devices, you can copy exported CSV or use a hosted version.
Notes:
- Prototype uses Gujarati labels and Supervisor prefilled as Anad Halpati.
- Sites preloaded: Galuda talav, Malekpor house, Malekpor toilet.
- Materials preloaded: Cement, Steel 8/10/12/16mm, Paver 60mm, Paver pressing kapachi.
- Suppliers: Patel Cement Traders, Bhavani Steel Works, Riddhi Transport, Local Supplier.
